# blog.krofn.online

Personal blog interface prototype built around a dark editorial layout.

> **Status:** Inactive prototype.  
> Kept public as a portfolio experiment, not as an active publication.

## Overview

The project explores a personal writing website with:

- public home page
- featured and latest posts
- year-based post filtering
- Supabase-backed post data
- rich text editor dependencies
- dark editorial UI

## Stack

- Next.js
- TypeScript
- React
- Supabase
- Tiptap
- Tailwind CSS
- Lenis
- Zod

## Run locally

```bash
npm install
npm run dev
```

or:

```bash
bun install
bun run dev
```

Open `http://localhost:3000`.
